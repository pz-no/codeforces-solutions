# Codeforces

This repository includes my programming solutions to the algorithm challenges from [Codeforces](https://www.codeforces.com). All of them have been submitted to
and accepted by the online jury. I am continuously uploading the solutions in a regular basis. Feel free to clone or fork the repository for your
purpose. I publish the code under [MIT license](LICENSE). The solutions are imperfect, so you can reach me by issuing a ticket. I am open to
comments and questions.
